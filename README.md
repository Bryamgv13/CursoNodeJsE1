# Primera entrega - Fundamentos Node Js


## Requerimientos
1.	node js
2.  npm

## Construccion y pruebas
1.  Clonar proyecto
```
git clone https://github.com/Bryamgv13/CursoNodeJsE1.git
```
2.  Instalar Dependencias
```
npm install
```
3.  Correr servidor
```
npm start
```
